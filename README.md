# 🍅 싱싱고 프로젝트: 식품 유통기한 관리를 혁신하는 웹 애플리케이션

> [싱싱고 바로가기](https://sslee1210.github.io/singsinggo/)

## 🎯 프로젝트의 비전
"리액트를 활용하여 사용자 중심의 식품 유통기한 관리 시스템을 구현하고, 팀워크를 통한 기획부터 구현까지의 전 과정을 성공적으로 완성하는 것"

## 💡 핵심 기능과 가치
- **냉장고 관리:** 식품 보관 및 유통기한 확인 기능을 통해 사용자가 식품 낭비 없이 관리할 수 있도록 돕는다. 이를 통해 환경 보호에 기여하며, 사용자의 경제적 부담을 줄인다.
- **장바구니 기능:** 식품 구매 계획을 체계적으로 관리함으로써 사용자의 식품 구매 효율성을 향상시킨다. 이는 사용자의 생활 편의성을 증대시키며, 건강한 식습관을 유도한다.
- **달력 기능:** 보관한 식품의 유통기한을 한눈에 파악할 수 있도록 돕는다. 이를 통해 사용자가 식품을 안전하게 섭취할 수 있도록 하며, 사용자 경험을 극대화한다.

## 🛠 사용 기술 및 도구
- **프로젝트 관리 도구:** Github, Figma, Notion, Google spread sheet를 활용하여 팀 간의 작업 분배 및 진행 상황을 체계적으로 관리하였다. 이를 통해 팀원 간의 원활한 커뮤니케이션을 가능하게 하였다.
- **개발 도구:** Visual Studio Code와 Adobe Illustrator를 사용하여 효율적인 코드 작성 및 디자인 작업을 수행하였다.
- **웹 프레임워크:** REACT를 활용하여 현대적이고 동적인 웹 애플리케이션을 구현하였다. 이를 통해 사용자에게 빠른 반응 속도와 편리한 인터페이스를 제공하였다.
- **스타일링:** .module.css를 사용해 사용자 친화적이고 세련된 UI를 제작하였다. 이로써 사용자에게 보다 편안한 사용 경험을 제공하였다.

## 👥 팀원 및 역할
- 이성수: **프로젝트 리더, 주요 기능 구현** - 리더로서 팀을 이끌고, 주요 기능 구현에 적극적으로 참여하며 프로젝트의 성공적인 완성을 이끌었다.
- 성영은: **프로젝트 진행 기록, 기능 구현** - 프로젝트의 전반적인 진행 상황을 체계적으로 기록하며, 핵심 기능 구현에 참여하였다. 이를 통해 팀원들이 프로젝트의 진행 상황을 쉽게 파악할 수 있도록 도왔다.
- 정하솔: **Github 관리, 디자인 총괄** - 코드 버전 관리 및 전체적인 디자인 품질을 책임지었다. 이를 통해 프로젝트의 안정적인 진행 및 고품질의 결과물 제작을 가능하게 하였다.
- 조유빈: **디렉토리 관리, 기능 및 디자인 구현** - 코드의 효율적인 관리를 위해 디렉토리를 체계적으로 관리하였다. 또한 기능 및 디자인 구현에 참여하여 프로젝트의 완성도를 높였다.

## 🚀 프로젝트 진행 및 성과
1. **기획:** 사용자 중심의 서비스 개발을 위해 체계적인 기획을 수립하였다. 이를 통해 사용자의 실제 요구 사항을 반영한 서비스를 제공하였다.
2. **개발 계획:** 효율적인 프로젝트 진행을 위해 구체적인 개발 계획을 수립하였다. 이를 통해 각 팀원의 역할을 명확히 하고, 작업의 우선순위를 결정하였다.
3. **UI 디자인:** 사용자 친화적인 인터페이스 디자인을 통해 사용자 경험을 향상시켰다. 이로써 사용자가 서비스를 보다 쉽고 편리하게 이용할 수 있도록 하였다.
4. **기능 구현:** 핵심 기능의 안정적인 구현을 통해 사용자의 식품 관리 효율성을 향상시켰다. 이는 사용자의 생활 편의성 증대 및 건강한 식습관 형성에 기여하였다.
5. **스타일링:** 세련된 디자인을 통해 애플리케이션의 품질을 향상시켰다. 이로써 사용자에게 긍정적인 인상을 주고, 사용자 경험을 더욱 향상시켰다.

- **아쉬웠던 점:** 리액트 활용에 대한 초기 어려움이 있었지만, 이를 극복하며 리액트에 대한 깊은 이해와 활용 능력을 향상시켰다.
- **성취감:** 팀원들과의 원활한 커뮤니케이션 및 협력을 통해 프로젝트를 성공적으로 완성하였다. 이 과정에서 팀워크의 중요성과 협업의 가치를 체감하였다.
- **배운 점:** 리액트와 관련한 기술적 이해도를 상승시키고, 팀원들과의 원활한 커뮤니케이션을 통한 협업 능력을 향상시켰다. 또한 밝은 팀 분위기를 유지하고, 팀원들을 이끌 수 있는 리더십 능력을 키웠다.
